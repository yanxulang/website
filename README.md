# 言序官网

[![CI](https://github.com/YanXuLang/website/actions/workflows/ci.yml/badge.svg)](https://github.com/YanXuLang/website/actions/workflows/ci.yml)
[![Pages](https://github.com/YanXuLang/website/actions/workflows/pages.yml/badge.svg)](https://github.com/YanXuLang/website/actions/workflows/pages.yml)

言序官方网站，由 React、TypeScript、Tailwind CSS 与 Vite 构建，并独立发布到 GitHub Pages。

- 官网：https://yanxulang.github.io/website/
- 文档：https://yanxulang.github.io/docs/
- 下载中心：https://yanxulang.github.io/website/download/

## 开发

```sh
npm install
npm run check
npm run build
node check-site.mjs dist
```

语言核心、文档与 VS Code 扩展分别位于 `YanXuLang/language`、`YanXuLang/docs` 与 `YanXuLang/vscode`。
